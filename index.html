Welcome to Our Website - Version 4

This website features a sophisticated two-tier navigation menu system with enhanced accessibility and keyboard navigation support. The top row contains three main categories: Products, Services, and Company.

New in Version 4:

  • Mobile Hamburger Menu: On mobile devices (≤768px), the main navigation switches to a slide-in hamburger menu with accordion-style sections
  • Multiple Accordion Sections: On mobile, you can expand multiple category sections simultaneously for easier browsing and comparison
  • Persistent Secondary Menu: The secondary navigation bar (Home, Shop, Deals, etc.) remains visible and horizontally scrollable on mobile
  • Adaptive Navigation UX: Desktop/tablet uses hover dropdowns, mobile uses touch-optimized hamburger menu
  • Improved Mobile Spacing: Content properly positioned below fixed navigation on all screen sizes
  • Full Accessibility: Hamburger menu includes ARIA attributes, keyboard navigation, and focus management
  • Skip Link: Keyboard users can press Tab once on page load to reveal a "Skip to main content" link that bypasses all navigation (WCAG 2.1 Level A compliance)

Preserved from Version 3:

  • Fixed Menu Overlay: Entire menu system fixed at top of viewport, overlays page content - zero layout shift!
  • Backdrop Blur Effect: Page content gets subtle blur and darkening when dropdown/menu opens
  • Keyboard Navigation: Full keyboard support with Tab, Enter/Space, and Escape keys
  • ARIA Attributes: Comprehensive screen reader support
  • CSS Custom Properties: Single source of truth for timing and color values
  • Reduced Motion Support: Respects user's prefers-reduced-motion setting

Accessibility: Press Tab when the page loads to reveal the "Skip to main content" link, allowing you to bypass all navigation and jump directly to the page content. This is especially helpful for keyboard and screen reader users.

Desktop/Tablet (>768px): Hover over any top menu item to reveal an expandable dropdown panel with three columns. The second menu row smoothly slides down beneath the revealed content.

Mobile (≤768px): Tap the hamburger menu button to open a slide-in navigation panel. Tap category headers to expand accordion sections. Multiple sections can remain open simultaneously for easier browsing. The secondary menu (Home, Shop, Deals, Blog, Support, My Account, Wishlist) remains visible and accessible via horizontal scrolling.
